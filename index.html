00.5 / What is PCR-QR?

PCR-QR 到底是什么？

PCR-QR 是一个双层公共问题诊断框架。外层 PCR 分析公共结构如何制造问题；内层 QR 分析问题如何在人的身体、感知、情绪和日常节律里显影。v4.0 再加入逻辑生成学：检查这个问题如何被命名、被加压、被滑义、被相变，并最终转化成接口工程。

PCR-QR is a dual-layer diagnostic framework for public problems. The outer PCR layer analyzes how public structures produce a problem; the inner QR layer analyzes how that problem manifests in the body, perception, emotion, and daily rhythm. v4.0 adds Logic Generative Studies: it audits how the problem is named, pressured, semantically shifted, phase-transitioned, and finally transformed into interface engineering.

一句话：
结构如何变成身体感受？
In one sentence:
How does structure become bodily experience?

PCR-QR 不满足于说“政策不好”“意识落后”“个体痛苦”。它要把三者接起来：公共结构如何通过语言、制度、通道和日常节律进入人的身体；人的身体经验又如何反过来暴露结构堵点。

PCR-QR does not stop at saying “policy is bad,” “consciousness is outdated,” or “individuals suffer.” It connects them: how public structures enter bodies through language, institutions, channels, and daily rhythm; and how embodied experience reveals structural blockage in return.

02 / Logic Generative Core

语言操作层：四步不是装饰，是引擎

命名、加压、滑义、相变不只是分析标签，而是让公共问题从“表面议题”进入“生成条件”的四级引擎。

Naming, Pressure, Drift, and Phase Transition are not decorative analytic labels. They are the four-stage engine that moves a public problem from surface issue to generative condition.

01

命名行为 / Naming

审计问题被叫什么。命名不是标签，而是规则修改。谁命名，谁就预设了行动路径。

Audit what the problem is called. Naming is not labeling; it modifies the game rule. Whoever names pre-sets the action path.

rule modification
02

加压行为 / Pressure

追问这个词如何被重复、合法化、制度化，直到隐藏权力和堵塞通道显形。

Ask how the term is repeated, legitimized, and institutionalized until hidden power and blocked channels appear.

critical pressure
03

滑义行为 / Drift

让词跨层移动。医学词、政策词、伦理词、经济词、身体词经常互相偷换责任。

Let terms move across layers. Medical, policy, ethical, economic, and bodily terms often displace responsibility onto one another.

semantic drift
04

把问题从“谁错了 / 如何修补”相变为“哪个生成条件持续制造这个问题”。

Shift from “who is wrong / how to fix” to “which generative condition keeps producing this problem.”

system transition

04 / PCR-QR Core

双层结构保留，但成为运行接口

PCR-QR 的核心仍然是外层公共结构与内层身体经验的连接。v4.0 的不同在于：连接不是解释，而是接口；接口不是展示，而是现实操作入口。

PCR-QR still connects outer public structure with inner bodily experience. v4.0 changes the status of that connection: it is not explanation but interface; not display but operational entrance.

P

Policy / 政策

法律、制度、预算、执行机制。问：什么被正式允许、禁止、资助、惩罚？

Law, institutions, budget, enforcement. What is formally allowed, prohibited, funded, or punished?

C

Consciousness / 文明意识

什么算问题，谁能说话，什么语言被承认，什么经验被取消。

What counts as a problem, who can speak, which language is recognized, which experience is cancelled.

R

Routine / 生活规律

工作、家庭、消费、居住、算法与日常流程如何重复生产问题。

How work, family, consumption, residence, algorithms, and daily routines reproduce the problem.

Q

Qualia / 感质

问题在身体里的纹理：强度、分布、压强、麻木、恐惧、疲惫、羞耻、过载。

The bodily texture of a problem: intensity, distribution, pressure, numbness, fear, fatigue, shame, overload.

R

Rhythm / 节律

感质被采样和显影的时间结构：连续/事件、快/慢、代际、政策周期、媒体周期。

The temporal structure through which qualia is sampled and manifested: continuous/event-based, fast/slow, intergenerational, policy-cycle, media-cycle.

Interface Equation: C = Q × R / 文明意识层的实际运行机制，是感质在节律中显影。
